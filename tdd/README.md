# Test-Driven Development (TDD) Suite

Comprehensive tools for maintaining high code quality through rigorous testing.

## Skills in this Category
- **[tdd-architecture-patterns](./tdd-architecture-patterns)**: Designing testable software systems using established architectural patterns.
- **[tdd-automation-flow](./tdd-automation-flow)**: Optimizing the developer feedback loop with automated testing triggers and watchers.
- **[tdd-bdd-gherkin](./tdd-bdd-gherkin)**: Using behavior-driven development and Gherkin syntax to align business goals with technical tests.
- **[tdd-cycle-manager](./tdd-cycle-manager)**: Guides established workflows through the Red-Green-Refactor cycle of Test-Driven Development.
- **[tdd-legacy-rescue](./tdd-legacy-rescue)**: Strategies for bringing legacy code under test and refactoring it safely.
- **[tdd-mocking-isolation](./tdd-mocking-isolation)**: Techniques for isolating units under test using mocks, stubs, and fakes.
- **[tdd-property-testing](./tdd-property-testing)**: Using generative testing to explore large input spaces and find obscure bugs.
- **[tdd-refactor-expert](./tdd-refactor-expert)**: Guidance and checklists for the Refactor phase of TDD, ensuring clean code without breaking tests.
- **[tdd-test-generator](./tdd-test-generator)**: Automatically generates test boilerplate, edge cases, and unit tests based on function specifications.
