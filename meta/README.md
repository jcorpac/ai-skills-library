# Meta-Skill & Orchestration

Advanced strategies for managing large skill ecosystems and context.

## Skills in this Category
- **[meta-agentic-chunking](./meta-agentic-chunking)**: Breaking complex tasks into discrete sub-steps with isolated context for maximum precision.
- **[meta-context-budgeting](./meta-context-budgeting)**: Techniques for managing the context window limit in long-running AI agents.
- **[meta-mcp-dynamic-tools](./meta-mcp-dynamic-tools)**: Using the Model Context Protocol to dynamically manage agent capabilities and context.
- **[meta-recursive-discovery](./meta-recursive-discovery)**: Enabling agents to autonomously search for and adopt new specialized skills.
- **[meta-skill-importer](./meta-skill-importer)**: Automated workflows for importing and adapting external skills from third-party repositories (e.g., Google Gemini Skills) into your local library.
- **[meta-skill-routing](./meta-skill-routing)**: Strategies for dynamic skill discovery to minimize context overhead.
- **[skill-author](./skill-author)**: Comprehensive guide and automation for creating, maintaining, and packaging AI skills within the skills library. Use when adding new skills or updating existing ones to ensure they meet repository standards and are correctly packaged for distribution.
- **[template-skill](./template-skill)**: A template skill demonstrating the required structure and metadata for a skill package.
