# ui-asset-generation

## Purpose
Provides the visual "soul" of the software for **Paintingway**. Details the prompt engineering, parameter selection, and color-matching guidelines necessary to use `image_gen_v3` to create aligned 3D assets, textures, and iconography.

## Usage
1. Analyze the primary layout and palette provided by the design schema.
2. Structure detailed visual prompts using the `image_gen_v3` shared tool.
3. Incorporate strict color theory (hex mapping), whimsical design patterns, and lighting rules.
4. Curate the generated textures or icons directly into the final application context for seamless integration.
