# Git Mastery Suite

Professional version control and repository management.

## Skills in this Category
- **[git-atomic-commits](./git-atomic-commits)**: Principles for making small, focused, and reversible changes to a repository.
- **[git-branching-flow](./git-branching-flow)**: Choosing and implementing the right collaboration strategy for your team's development speed.
- **[git-conventional-naming](./git-conventional-naming)**: Implementing the Conventional Commits specification for structured and machine-readable commit messages.
- **[git-github-master](./git-github-master)**: Professional strategies for publishing, collaborating, and managing repositories on GitHub.
- **[git-history-polish](./git-history-polish)**: Advanced techniques for cleaning up and organizing your local history before pushing to a shared repository.
- **[git-hooks-automation](./git-hooks-automation)**: Using client-side Git hooks to automate testing and linting before code is shared.
- **[git-tagging-releases](./git-tagging-releases)**: Managing software versions and releases using Semantic Versioning (SemVer) and Git tags.
